## 两两交换链表中的节点

这个指来指去的操作步骤需要好好想一下

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func swapPairs(head *ListNode) *ListNode {
    if head == nil{
        return head
    }
    dummy := new(ListNode)
    dummy.Next = head
    prev := dummy
    cur := head

    for cur != nil && cur.Next != nil{
        // prev指向cur的下一个
        prev.Next = cur.Next
        nextNext := cur.Next.Next
        cur.Next.Next = cur // 指向cur
        cur.Next = nextNext // cur指向next的next
        prev = cur
        cur = cur.Next//将指针后移
    }
    return dummy.Next
}
```

## 删除链表的倒数第 N 个结点

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func removeNthFromEnd(head *ListNode, n int) *ListNode {
    // 快指针先走n+1步，然后慢指针才开始走
    dummy := new(ListNode)
    dummy.Next = head

    slow, fast := dummy,dummy
    for i:=0;i<n+1;i++{
        if fast != nil{
            fast = fast.Next
        }
    }
    // 一起走
    for fast != nil{
        fast = fast.Next
        slow = slow.Next
    }
    // 找到了
    nextNext := slow.Next.Next
    slow.Next.Next = nil
    slow.Next = nextNext
    return dummy.Next
}
```

## 链表相交

这就必须考虑到两个链表的长度了，一旦在某个节点上相交，那么共用后面的节点，两个链表的差异就体现在相交前的节点数量上了。

所以必须要遍历两个链表得出长度差值，然后让长的那一个先跳过差值，这样两个链表就有同样数量的剩下的节点了，就可以同时往后遍历寻找相等的节点。

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func getIntersectionNode(headA, headB *ListNode) *ListNode {
    curA := headA
    cntA := 0
    for curA != nil{
        cntA ++
        curA = curA.Next
    }
    curB := headB
    cntB :=0
    for curB != nil{
        cntB ++
        curB = curB.Next
    }

    diff := 0
    if cntA > cntB{
        diff = cntA-cntB
        for curA = headA;diff > 0;diff--{
            curA = curA.Next
        }
        curB = headB
    }else{
        diff = cntB-cntA
        for curB = headB;diff>0;diff--{
            curB = curB.Next
        }
        curA = headA
    }
    for curA != curB{
        curA = curA.Next
        curB = curB.Next
    }
    return curA

}
```

## 环形链表

要返回环的第一个起点，就是确认有环之后，slow指针从头出发，fast指针从相遇节点出发，但是fast也是只走1步，再次相遇就是环的起点。

```go
/**
 * Definition for singly-linked list.
 * type ListNode struct {
 *     Val int
 *     Next *ListNode
 * }
 */
func detectCycle(head *ListNode) *ListNode {
    slow, fast := head,head
    if head == nil{
        return nil
    }
    for fast.Next!=nil &&fast.Next.Next != nil{
        fast = fast.Next.Next
        slow = slow.Next
        if fast == slow{
            break
        }
    }
    if fast.Next == nil || fast.Next.Next == nil{
        return nil
    }
    slow = head
    for slow != fast{
        slow = slow.Next
        fast = fast.Next
    }
    return slow
}
```
